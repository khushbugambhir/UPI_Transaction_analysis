UPI Transaction Analysis Dashboard

Overview :-

This project presents an interactive UPI Transaction Analysis Dashboard designed to analyze digital payment trends across different dimensions such as time, location, payment methods, and user demographics.
The dashboard helps in understanding transaction patterns and financial behavior using visual analytics.

Objectives :-

1)Analyze UPI transaction trends over time (monthly analysis).

2)Compare transaction amounts and balances.

Key Insights :-

1)Transaction volume fluctuates monthly with peaks in mid-year.

2)Certain cities show higher transaction activity.

3)Digital payment methods dominate usage.

4)Balance trends indicate spending behavior patterns.

Tools & Technologies Used :-

Power BI 

Microsoft Excel (for dataset)
